# SecureShell — Encryption & Decryption Web App

*SecureShell* is a front-end web application for performing client-side encryption and decryption using Base64, DES, and AES. It also features a chatbot for interactive queries, a feedback form, and a professional portfolio section.

---

##  Features

- *Encryption & Decryption Tool*
  - Base64 encoding/decoding
  - DES encryption/decryption
  - AES encryption/decryption
- *Interactive Chatbot*
  - Can answer questions about encryption, decryption, ciphers (AES, DES, Base64)
  - Responds to greetings and general knowledge queries
- *Feedback Form*
  - Submit feedback directly through the website
- *Professional Portfolio*
  - Skills, areas of interest, and certifications displayed
- *Debug/Log Section*
  - Shows real-time log messages for troubleshooting
- *Responsive Design*
  - Works on desktops, tablets, and mobile devices

##  Usage

1. Clone the repository or download the ZIP file:
   ```bash
   git clone https://github.com/sundas8124/SecureShell.git

2. Open index.html in your browser.


3. Use the encryption/decryption tool by entering text, selecting a cipher, and entering a key.


4. Interact with the chatbot by clicking the "Chat" button at the bottom-right corner.


5. Check the "Debug/Log" section for troubleshooting messages.


6. Explore the professional portfolio section at the bottom of the page.
